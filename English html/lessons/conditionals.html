Conditionals in English

Master the different types of conditional sentences

Zero Conditional

Used to talk about general truths and scientific facts. The structure is:

  • If/When + present simple, present simple
  • Expresses general truths
  • Describes automatic or scientific results

Examples:

  • If you heat water to 100°C, it boils.
  • When it rains, the ground gets wet.
  • If you don't eat, you get hungry.

First Conditional

Used for possible situations in the future. The structure is:

  • If + present simple, will + infinitive
  • Expresses real possibilities
  • Talks about likely future events

Examples:

  • If it rains tomorrow, I will stay at home.
  • If you study hard, you will pass the exam.
  • I will help you if you need me.

Second Conditional

Used for unlikely or imaginary situations. The structure is:

  • If + past simple, would + infinitive
  • Expresses unlikely situations
  • Describes hypothetical situations

Examples:

  • If I won the lottery, I would travel the world.
  • If I had more time, I would learn another language.
  • What would you do if you were president?
